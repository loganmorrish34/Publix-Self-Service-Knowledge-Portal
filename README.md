# Publix IT Self Service Portal

This is a simple React and Vite project for a Publix styled IT self service portal.

## Features
- Green Publix inspired design
- Dark mode toggle
- Employee sign in panel
- Searchable knowledge articles
- Incident and request reporting form
- Sample success message for demo use

## How to run locally
1. Install Node.js
2. Open the project folder in a terminal
3. Run `npm install`
4. Run `npm run dev`

## How to deploy on Vercel
1. Upload this whole folder to a GitHub repo
2. In Vercel choose Import Project
3. Select the repo
4. Vercel should detect Vite automatically
5. Click Deploy
